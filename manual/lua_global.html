Lua API Reference

Doxyrest exports all the Doxygen XML database in form of inter-connected Lua variables. It also exports a couple of handy functions useful for .rst file emission.

In addition to the globals above, all the variables passed via command line using --define will also be added to the Lua global namespace.

Detailed Documentation

function includeFile (frameFileName)

Process the frame in frameFileName and includes its output at the current position of the current frame 1.

function generateFile (targetFileName, frameFileName)

Generates a new file named targetFileName by processing the frame frameFileName and writing its output to targetFileName 1.

g_globalNamespace

Holds a Compound table containing global namespace of the source tree.
