Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Enumerated potential types for APR process locking methods Check APR_HAS_foo_SERIALIZE defines to see if the platform supports APR_LOCK_foo. Only APR_LOCK_DEFAULT is portable.
